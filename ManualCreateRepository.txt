1. Создаём на диске ? папку для проектов "projects".
2. В папке "projects", создаём папку для проекта "?".
3. На GitHab создаём репозиторий с таким же названием, как в пункте 2. (Снимаем все галочки)
4. Генерируем папку ".git"
	git init
5. Делаем свой первый коммит (ниже описанные пункты локальные): 
	1) Создаём 2 файла ".gitignore" и "README.md".
	2) Делаем файлы видимыми для Git, для дальнейшего записи в коммит.
		git add название_файла.формат
		git add . //или добавить все файлы сразу 
	3) Проверяем статус файлов
		git status
	4) Делаем коммит 
		git commit -m "Initialisation"
6. Синхронизируем локальную папку с проектом с удалённой репозиторием на GitHub (делается один раз, для каждого проекта)
	git remote add origin https://github.com/SokolDDev/shuffle-words-game.git
7. Пушим
	git push -u origin main
